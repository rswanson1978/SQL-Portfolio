# Preoperative Clinical Research Pipeline

## Overview

This project demonstrates the development of a longitudinal research dataset
for patients receiving preoperative evaluation. Separate clinical domains were
extracted and retained as independent tables so each component could be
validated and modified as research requirements evolved.

## Data Domains

- Preoperative encounters
- Encounter diagnoses
- Problem-list diagnoses
- Surgical and anesthesia cases
- Postoperative complications
- Delirium-related flowsheet assessments
- Clinical notes

## Key SQL Techniques

- Temporal joins
- Multi-domain EHR integration
- Reference-table lookups
- Structured and unstructured data extraction
- Independent analytic table creation
